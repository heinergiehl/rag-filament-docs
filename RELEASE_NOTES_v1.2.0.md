# Release Notes v1.2.0

Release date: 2026-04-02

This release focuses on polish and reliability improvements in the widget and demo environment.

## Highlights

- Improved widget auto-scroll during streaming so long answers stay visible while content is still arriving.
- Improved reference-card contrast across all widget themes in both light and dark mode.
- Fixed live demo asset loading issues caused by a cross-origin `ASSET_URL` mismatch.

## Buyer Impact

- Better chat usability during long streaming responses
- Cleaner and more readable source/reference blocks
- More stable live demo behavior when evaluating the plugin

## Upgrade Notes

- No intended breaking changes
- Existing integrations can update normally
