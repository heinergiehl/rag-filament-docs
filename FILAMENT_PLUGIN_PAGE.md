## What This Plugin Adds

Filament RAG adds a managed chatbot layer to a Laravel + Filament app:

- bot management inside Filament
- RAG source ingestion and retrieval controls
- embeddable public chat widgets
- conversation visibility inside the admin panel
- queue-aware ingestion, retries, and operational workflows

It helps you ship support assistants, internal copilots, onboarding bots, or domain-specific help without building the admin layer from scratch.
