# Release Notes v1.1.0

## Added

- HTML and PDF ingestion support alongside existing text, markdown, and URL workflows.
- Token-aware chunk sizing controls for better ingestion tuning.
- Rich Markdown answer rendering in the widget and history views.
- Stronger citation UX with inline references and verification panels.
- More widget customization with font presets and branded presentation controls.

## Changed

- HTML ingestion now extracts readable article content instead of flattening raw markup.
- Chunking now keeps more document structure and heading context.
- Source upload and ingestion UX better reflects the supported source types.
- Response rendering now supports richer formatted answers for end users.

## Documentation

- Added dedicated concept pages for bots, RAG sources, ingestion and retrieval, widget behavior, context areas, and conversations.
- Added a reference-links page so precise concept links can be returned by the public demo bot.
- Expanded docs coverage around how to create and customize bots and how to create and ingest sources.
